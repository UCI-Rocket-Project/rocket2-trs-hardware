# TRS-ECU — Power

## Power Source

TRS-ECU is powered from an **8.4V LiPo battery** or regulated 8.4V bench supply located in the bunker.

```
[8.4V Power Supply] ──► [AP63205WU Buck] ──► +5V ──► [AMS1117-3.3] ──► +3.3V
```

Refer to [TRS-GND Power](../trs-gnd/power.md) for full regulator specs, decoupling capacitor values, and power select circuit details — the power architecture is identical across all TRS units.

## TRS-ECU Power Notes

## Power-On Checklist

- [ ] 8.4V supply voltage confirmed before connecting
- [ ] 3.3V LED (D3) illuminated on power-up

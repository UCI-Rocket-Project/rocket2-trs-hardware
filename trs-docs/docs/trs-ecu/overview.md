# TRS-ECU — Overview

**Location:** Bunker  
**Role:** Receives live telemetry data from ECU

---

## Description

TRS-ECU is located in the bunker and receives live telemetry from the ECU over a dedicated radio link. 

TRS-ECU is a separate unit from TRS-GND because there is only 2 radio modules on the TRS board.

---

## Key Responsibilities

- Receive live telemetry from the ECU over radio
- Forward telemetry onto the bunker LAN for the GSE laptop and GUI PC
- Operate continuously from bunker power throughout the test campaign
